01 — Sistema

Inteligencia que lo
impregna todo

AETHER es un sistema de inteligencia de riesgos globales que corre completamente en local, sin servidor, sin base de datos, sin dependencias de backend. Un único archivo HTML con todo el motor analítico embebido.

Cada país, cada categoría, cada correlación están vivos — el algoritmo propaga shocks en cascada a través del tiempo, detecta puntos de inflexión antes de que sean evidentes y genera distribuciones de probabilidad mediante simulación estocástica.

La IA no es un chatbot genérico. Es Mistral corriendo en tu máquina, con contexto completo de la región activa, generando predicciones estructuradas, escenarios y análisis sin enviar datos a ningún servidor externo.

02 — Algoritmo

Motor de
cinco capas

01
Score Base
Datos históricos calibrados con World Bank, Freedom House, Fragile States Index, GHS Index y 15+ organismos internacionales. Actualizable en tiempo real vía World Bank API.
score_base = f(HDI, FSI, Freedom, SIPRI, GHS, WB_indicators)
02
Vector de Velocidad
Cada categoría por país tiene un vector de tendencia — no solo el estado actual sino la dirección y magnitud del cambio. Un país con score 45 subiendo a +0.8/semana es más peligroso que uno con 65 estable.
score_vel = base + (base × velocity × 0.28)
03
Boost de Correlación
Matriz de correlación 16×16. Cuando categorías fuertemente correladas están simultáneamente en zona de riesgo, el score se amplifica. Economía + Finanzas elevadas = más peligroso que cada una por separado.
score += Σ(CORR[i][j] × score_j × weight) if score_j > threshold
04
Propagación en Cascada
Cada par de categorías tiene {strength, lag (semanas), decay}. Un shock climático tarda 4 semanas en impactar la seguridad alimentaria con 80% de intensidad, atenuándose 5%/semana. La cascada genera impactos de segundo y tercer orden.
shock[t+lag] = magnitude × strength × (1 − resilience×0.4) × (1−decay)^t
05
Monte Carlo & Resiliencia
400 simulaciones con ruido gaussiano (Box-Muller) generan bandas de confianza p10/p25/p50/p75/p90. El Resilience Index modela la capacidad de amortiguación — países con alta ciencia, buena gobernanza y diversificación energética absorben mejor los shocks.
P(score_T ∈ [p10,p90]) = 0.80  |  resilience = Σ(cat_strength_inv × weight)

05 — Instalación

Cero dependencias.
Un archivo.

01
Instalar Ollama + Mistral
IA local opcional. Si no necesitas predicciones, puedes saltar al paso 3.
ollama pull mistral
  • Ollama para Windows / macOS / Linux
  • Mistral 7B (~4GB, corre en CPU o GPU)
02
Arrancar Ollama con CORS habilitado
Necesario para que el navegador pueda conectar con el servidor local.
# Windows PowerShell
$env:OLLAMA_ORIGINS="*"; ollama serve
# Linux / macOS
OLLAMA_ORIGINS="*" ollama serve
03
Servir AETHER localmente
Necesario para cargar el mapa D3.js desde CDN sin errores CORS.
cd /carpeta/con/AETHER
python -m http.server 8080
04
Abrir en el navegador
Chrome o Edge recomendado. Firefox también funciona.
http://localhost:8080/AETHER_v1.html
